## Prettier Config

1. => `Cmd + Shift + P` pour Mac, `Ctrl + Shift + P` pour Windows et Linux
2. => `>Preferences: Open User Settings (JSON)`
   ![](../../assets/open-user-settings.png)
3. =>
   ```json
   {
     // Formatting Settings
     "editor.defaultFormatter": "esbenp.prettier-vscode",
     "editor.formatOnSave": true
   }
   ```
