## box-sizing

```css
.border-box {
  box-sizing: border-box;
  width: 200px;
  padding: 20px;
  border-width: 10px;
}

.content-box {
  box-sizing: content-box;
  width: 200px;
  padding: 20px;
  border-width: 10px;
}
```
